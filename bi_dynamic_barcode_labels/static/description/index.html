Print Dynamic Barcode Labels app for All(Products,Templates,Sale,Purchase,Picking) in Odoo

This Odoo apps module helps to print all sort dynamic barcode labels from Products, Templates, Sales, Purchase and Pickings for printing barcode in Odoo.This Odoo apps helps to generate label according to company specification, every company has its own label size standard. Here with this help user can have option for customized label size template feature which is also configurable. User have option to enable/disable product price from barcode configuration. Based on configuration of barcode label height and width barcode label allows to prints from Products Variants ,Product Templates,Sales Orders,Purchase Orders and Picking(Delivery Order and Shipment). Also it has most useful features to print multiple labels on single page as per the quantity allocated on Orders or on Print section wizard.

Dynamic Label Size.

You can enter size to print dynamic product label.

Multi Products Label.

You can select multiple product to print multiple barcode in single click.

Barcode label Quantity from Order Quantity.

Get barcode label quantity to print from order quantity, you may change quantity.

Support Multi Barcode Type.

Print multi barcode type(ex. EAN13, EAN8, Code128, QR) on barcode label that you want.

Barcode Configuration

Under Inventory > Configuration > Setting you can set Barcode Configuration.

You have option to enable 'Hide Price' checkbox to hide the price from barcode labels.

Barcode labels Menu

Under Action menu you can see " Barcode Product Labels " on Product, Sale, Purchase, and Picking.

Barcode Product Labels Wizard

When you click on Barcode Product labels menu new wizard will open with default barcode quantity from order quantity also you can update the barcode quantity.

You can see printed barcode label of selected product with hidden price.

Barcode Configuration

Under Inventory > Configuration > Setting you can set Barcode Configuration.

You have option to disable 'Hide Price' checkbox to show the price from barcode labels.

Barcode Product Labels on Sales

You can see printed barcode label of selected product with price.
