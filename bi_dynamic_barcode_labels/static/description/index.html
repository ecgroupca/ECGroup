Print Dynamic Barcode Labels app for All(Products,Templates,Sale,Purchase,Picking) in Odoo

This Odoo apps module helps to print all sort dynamic barcode labels from Products, Templates, Sales, Purchase and Pickings for printing barcode in Odoo.This Odoo apps helps to generate label according to company specification, every company has its own label size standard. Here with this help user can have option for customized label size template feature which is also configurable. Based on configuration of barcode label height and width barcode label allows to prints from Products Variants ,Product Templates,Sales Orders,Purchase Orders and Picking(Delivery Order and Shipment). Also it has most useful features to print multiple labels on single page as per the quantity allocated on Orders or on Print section wizard.

Features

Dynamic Label Size.

You can enter size to print dynamic product label.

Multi Products Label.

You can select multiple product to print multiple barcode in single click.

Barcode label Quantity from Order Quantity.

Get barcode label quantity to print from order quantity, you may change quantity.

Support Multi Barcode Type.

Print multi barcode type(ex. EAN13, EAN8, Code128, QR) on barcode label that you want.
