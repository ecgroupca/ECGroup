Send Low Stock Alert To Inventory Manager.

This module helps to Send Low Stock Alert To Inventory Manager.

Alert will be triggered to notify the user with above mentioned access right of the low stock levels.

Based on configuration Scheduler will be run every one hour to check low stock and notify to the user accordingly.

Notification And Email will be send to the relevent user.
