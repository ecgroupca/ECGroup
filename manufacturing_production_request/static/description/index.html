Product Manufacturing Request

Manufacturing Order from Manufacturing Request

This app allow to create Manufacturing Request for selected product and create Manufacturing Order from it.


Main Features:
  • Allow Manufacturing User to create Manufacturing Request and Confirm Request.
  • Manufacturing Manager to "Approve" and "Create Manufacturing Orders" of Manufacturing Request.
  • "Manufacturing Production Request" print PDF Report.
