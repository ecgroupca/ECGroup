Parallel Routing Steps

Improve Your Manufacturing Performance

By default, all routings in Odoo manufacturing are sequential, with each step requiring completing the previous.

For many manufacturing processes, this isn't optimal. Multiple steps should be done simultaneously to lower lead times.

This module helps break free from Odoo's restrictions and unlocks parallel routings.

Configuration

Security Made Simple

1. Open up a routing and enable parallel steps.

2. Add routing steps in any order as required.

3. Save the record. You will not be able to select a line as required for another step until it has been saved.

4. Add the required steps as required.

Notes

There should only be a single final step to a manufacturing process.

A routing should not be changed from standard to parallel while there are open manufacturing orders using that routing.
