Sales Stock Reservation

Product stock or inventory items booking or reservation mainly used to lock product stock while making sales order this app helps user to reserve sales stock and notify by email on stock reservation, also cancel after reserve stock, and can see different view like pivot view, bar graph, line chart, pie chart in sales and warehouse module in Odoo.

Reserve Sales Stock

User can reserve sale stock based on source and destination location and also cancel reserves sales stock.

Email Notification on Stock Reservation.

Users selected in notify user will get email notification on sales stock reservation.

Different Reserved Stock View.

User can see different reserved stock view like pivot view, bar graph, line chart, and pie chart.

Description

Configuration

Under Inventory > Configurations > Settings user have to select source location and destination location to reserve stock.



Under Inventory > Configurations > Shipping > User must disable SMS Confirmation.


Product Quantity Before Reserve Stock

User can see product on hand quantity before reserve stock.


Reserve Stock

User can reserve stock by clicking on "Reserve Stock" button on sale order which is visible only if "Enable Stock Reservation" is checked.



New wizard will open on clicking of "Reserve Stock" button, here user can see default sale order line details and select "Notify User" to get email and enter "Reserve Quantity" to reserve stock.


User can view and access created reserved stock from smart button "Reserved Stock" and also see reserves stock history from "Reserved Stock" page in sale order.



On clicking of smart button reserved stock tree view will open.


User can see form view of reserved stock details with source and destination stock location.


Product Quantity After Reserved Stock

Product on hand quantity will decreased on reservation of stock.


Reserved Stock Email

Notify user will get email on stock reservation done.


Cancel Reserved Stock

User can also cancel reserved stock by clicking on "Cancel Reserved Stock" if in "Reserved" state.


Reserved stock state will changed from "Reserved" to "Cancelled".


Product Qty After Cancel Reserved Stock

Product on hand quantity will increased on canceling reserved stock.


Reserved Stock Menu

Under Inventory > Reporting > Reserved Stock menu user can see created all reserved stock.
